# Project Summary
CoreNexus Technology Solution is an IT support platform designed for businesses in Rome, offering a comprehensive suite of services including network management, cybersecurity, and IT process automation. The platform features an advanced ticket management system, integrated chat support, and an intuitive admin dashboard to enhance IT operations. Recent updates have introduced Kubernetes and container orchestration services, facilitating IT infrastructure modernization. New functionalities include an audit page, optimized SEO strategies, and AI enhancements for better user interaction. The recent focus has been on improving the admin user management system, including a new delete functionality for companies with appropriate safeguards.

# Project Module Description
The project consists of several key functional modules:
- **Homepage**: SEO-optimized introduction showcasing the audit service with a centered navigation bar.
- **Services Section**: Overview of IT services, including:
  - **IT Maintenance Contracts**: Information on maintenance packages.
  - **IT Consultancy Services**: Professional guidance for businesses.
  - **IT Process Automation**: Custom automation solutions for efficiency.
  - **Project Management**: Tools for managing IT and cybersecurity projects.
  - **Networking and Infrastructure**: Setup and management services for networks.
  - **Cybersecurity Solutions**: Advanced measures utilizing Wazuh and Zabbix.
  - **Alarm and Surveillance Systems**: Collaborations for security solutions.
  - **Website and E-commerce Development**: Creation and management of online platforms.
  - **VoIP Solutions**: Support and configuration for VoIP systems, enhanced with AI capabilities.
  - **Kubernetes and Container Services**: Design and management of Kubernetes clusters, containerization with Docker, and microservices orchestration.
- **Audit IT Gratuito**: Details the free IT audit service, including benefits and a step-by-step process.
- **Contact Page**: Direct contact options via phone, email, and WhatsApp.
- **Remote Support**: Page for remote assistance using UltraViewer.
- **Technologies**: Overview of utilized technologies.
- **Service Areas**: Coverage details and response times.
- **Client Area**: Secure client login for ticket management, including ticket creation and real-time chat.
- **Admin Dashboard**: Management interface for tickets, users, and analytics, featuring user management, role-based access control, and a new delete functionality for companies with proper safeguards.
- **Legal Documentation**: Compliance information.
- **Cookie Consent**: Management component for cookies.
- **Setup Page**: Admin account creation feature.
- **SEO Enhancements**: Improved meta tags and content optimization.
- **Blog**: Articles on IT and cybersecurity, including AI-related content.
- **FAQ Sections**: Integrated with JSON-LD Schema for user assistance.
- **AI Integration**: New functionalities in VoIP solutions, including AI-driven features.
- **Cursor Follower**: Visual effect that follows the cursor with a trailing animation.
- **ScrollToTop Component**: Component that scrolls the page to the top on route changes.
- **Navigation Enhancements**: Improved alignment of the navigation bar for better user experience.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.

- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.
- **@prerenderer/renderer-puppeteer**: Library for static pre-rendering.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Pre-render the project:
   ```
   pnpm run prerender
   ```
4. Build and pre-render together:
   ```
   pnpm run build:prerender
   ```
