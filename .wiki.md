# Project Summary
CoreNexus Technology Solution is an IT support platform designed for businesses in Rome, providing a comprehensive suite of services including network management, cybersecurity, server support, VoIP solutions, and project management. The platform features a sophisticated ticket management system, integrated chat support, and a user-friendly admin dashboard, allowing businesses to effectively manage their IT and cybersecurity projects. Additionally, it showcases service coverage areas through a dedicated "Where We Are" page.

# Project Module Description
The project consists of several functional modules:
- **Homepage**: SEO-optimized introduction to IT services with interactive features.
- **Services Section**: Overview of IT services, including security systems and AI video verification.
- **Networking Services**: Page for Networking and Reti services, detailing benefits and packages.
- **Security Services**: Page for Sicurezza Informatica, covering cybersecurity solutions and packages.
- **System Support Services**: Page for Assistenza Sistemistica, including IT support and maintenance services.
- **Project Management**: Module for IT & Cyber Security Project Management, featuring service details, phases, and a centered contact form.
- **Alarm Systems and Video Surveillance**: Page detailing customized alarm and surveillance systems, including packages for residential, business, and industrial needs.
- **Where We Are**: Page detailing service coverage areas including specific zones and response times.
- **Client Area**: Secure login for clients to manage tickets.
- **Ticket Management**: Tools for ticket handling, real-time notifications, and support chat.
- **Admin Dashboard**: Module for managing tickets, viewing statistics, and responding to inquiries.
- **User Management**: Module for managing users and companies.
- **Legal Pages**: Compliance documentation including Privacy Policy and Terms of Service.
- **Cookie Banner**: Component for cookie consent with enhanced visibility.
- **Setup Page**: Feature for creating admin accounts without email verification.
- **SEO Enhancements**: Advanced meta tags, schema markup, and optimized content for better search engine visibility.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Lint the project:
   ```
   pnpm run lint
   ```
