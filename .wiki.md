# Project Summary
CoreNexus Technology Solution is an IT support platform aimed at businesses in Rome, offering a comprehensive range of services such as network management, cybersecurity, server support, VoIP solutions, project management, and IT process automation. The platform includes a robust ticket management system, integrated chat support, and an intuitive admin dashboard, enabling businesses to efficiently oversee their IT and cybersecurity requirements. Enhanced service coverage pages improve user navigation and access to services.

# Project Module Description
The project consists of several functional modules:
- **Homepage**: SEO-optimized introduction to IT services.
- **Services Section**: Overview of IT services, including:
  - **Contratti di Manutenzione**: IT maintenance contracts.
  - **Consulenza Sistemistica**: IT consultancy services.
  - **Automazione Processi IT**: Custom automation solutions for IT workflows.
  - **Project Management**: Module for managing IT & Cyber Security projects.
  - **Networking e Reti**: Networking and infrastructure services.
  - **Sicurezza Informatica**: Cybersecurity solutions featuring Wazuh and Zabbix monitoring systems.
  - **Allarme e Videosorveglianza**: Alarm and surveillance systems with partnerships with Hikvision, Ajax, and Risco.
  - **Siti Web & E-commerce**: Website and e-commerce development.
  - **Centralini VoIP**: VoIP solutions and configurations.
- **Tecnologie**: Detailed overview of technologies used, categorized into:
  - **Sistemi Operativi**: Linux Debian, RedHat, Windows Server.
  - **Virtualizzazione**: VMware, Proxmox, Hyper-V.
  - **Networking**: Cisco, Fortinet, MikroTik, pfSense, OPNsense.
  - **Backup & Disaster Recovery**: Veeam, Acronis, R-Drive Image.
- **Where We Are**: Service coverage areas and response times.
- **Client Area**: Secure login for clients to manage tickets.
- **Ticket Management**: Tools for ticket handling and support chat.
- **Admin Dashboard**: Management of tickets and statistics.
- **User Management**: Managing users and companies.
- **Legal Pages**: Compliance documentation.
- **Cookie Banner**: Cookie consent component.
- **Setup Page**: Admin account creation feature.
- **SEO Enhancements**: Advanced meta tags and optimized content for search engines.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Lint the project:
   ```
   pnpm run lint
   ```
