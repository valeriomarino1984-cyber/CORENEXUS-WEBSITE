# Project Summary
CoreNexus Technology Solution is an IT support platform aimed at businesses in Rome, offering a comprehensive suite of services such as network management, cybersecurity, and IT process automation. The platform includes a sophisticated ticket management system, integrated chat support, and a user-friendly admin dashboard to enhance IT operations. Recent updates have introduced Kubernetes and container orchestration services, enabling businesses to modernize their IT infrastructure. New features include a dedicated audit page, enhanced SEO strategies, and AI integrations for improved user interaction.

# Project Module Description
The project consists of several key functional modules:
- **Homepage**: SEO-optimized introduction with a glitch effect on the main header, featuring a section for the new audit service.
- **Services Section**: Overview of IT services including:
  - **IT Maintenance Contracts**: Information on maintenance packages.
  - **IT Consultancy Services**: Professional guidance for businesses.
  - **IT Process Automation**: Tailored automation solutions for efficiency.
  - **Project Management**: Tools for managing IT and cybersecurity projects.
  - **Networking and Infrastructure**: Setup and management services for networks.
  - **Cybersecurity Solutions**: Advanced measures utilizing Wazuh and Zabbix.
  - **Alarm and Surveillance Systems**: Collaborations for security solutions.
  - **Website and E-commerce Development**: Creation and management of online platforms.
  - **VoIP Solutions**: Support and configuration for VoIP systems, enhanced with AI capabilities.
  - **Kubernetes and Container Services**: Design and management of Kubernetes clusters, containerization with Docker, and microservices orchestration.
- **Audit IT Gratuito**: Page detailing the free IT audit service, including benefits, analysis areas, and a step-by-step process.
- **Contact Page**: Quick access for direct contact via phone, email, and WhatsApp.
- **Remote Support**: Dedicated page for remote assistance using UltraViewer.
- **Technologies**: Overview of utilized technologies.
- **Service Areas**: Details on coverage and response times.
- **Client Area**: Secure client login for ticket management with features like ticket creation and real-time chat.
- **Admin Dashboard**: Management interface for tickets, users, and analytics, including user management and role-based access control.
- **Legal Documentation**: Compliance information.
- **Cookie Consent**: Management component for cookies.
- **Setup Page**: Feature for admin account creation.
- **SEO Enhancements**: Improved meta tags and content optimization.
- **Blog**: Articles on IT and cybersecurity, including AI-related content.
- **FAQ Sections**: Integrated with JSON-LD Schema for better user assistance.
- **AI Integration**: New functionalities in VoIP solutions, including AI-driven features.
- **Cursor Follower**: Visual effect that follows the cursor with a trailing animation.
- **ScrollToTop Component**: Newly added component that scrolls the page to the top on route changes.
- **Navigation Enhancements**: Added "Back to Homepage" buttons on various pages.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.
- **@prerenderer/renderer-puppeteer**: Library for static pre-rendering.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Pre-render the project:
   ```
   pnpm run prerender
   ```
4. Build and pre-render together:
   ```
   pnpm run build:prerender
   ```
