# Project Summary
CoreNexus Technology Solution is an IT support platform designed for businesses in Rome, offering a comprehensive range of services such as network management, cybersecurity, and IT process automation. The platform includes an advanced ticket management system, integrated chat support, and a user-friendly admin dashboard. Recent enhancements feature a Google Business Profile reviews component to build visitor trust, improved SEO strategies, Kubernetes services, and AI functionalities for user management and error handling.

# Project Module Description
The project consists of several key functional modules:
- **Homepage**: SEO-optimized introduction with navigation.
- **Services Section**: Overview of IT services, including:
  - **IT Maintenance Contracts**
  - **IT Consultancy Services**
  - **IT Process Automation**
  - **Project Management**
  - **Networking and Infrastructure**
  - **Cybersecurity Solutions**: Enhanced content on Wazuh SIEM and Zabbix, with:
    - **Active Response**: Automatic IP blocking during attacks.
    - **Custom Dashboards**: Tailored dashboards for different roles.
  - **Alarm and Surveillance Systems**
  - **Website and E-commerce Development**
  - **VoIP Solutions**
  - **Kubernetes and Container Services**
- **Audit IT Gratuito**: Information on the free IT audit service.
- **Contact Page**: Direct contact options, including a Google Business Profile link and a new Google Reviews component.
- **Remote Support**: Page for remote assistance.
- **Technologies**: Overview of utilized technologies.
- **Service Areas**: Coverage details, now includes Google Maps link.
- **Client Area**: Secure client login for ticket management.
- **Admin Dashboard**: Management interface for tickets, users, and analytics.
- **Legal Documentation**: Compliance information.
- **Cookie Consent**: Management component for cookies.
- **Setup Page**: Admin account creation feature.
- **Blog**: Articles on IT and cybersecurity.
- **FAQ Sections**: Integrated with JSON-LD Schema.
- **AI Integration**: Enhanced functionalities in VoIP solutions.
- **Cursor Follower**: Visual effect that follows the cursor.
- **ScrollToTop Component**: Component that scrolls the page to the top on route changes.
- **Navigation Enhancements**: Improved navigation bar alignment.
- **Google Reviews Component**: Displays real-time Google reviews to enhance visitor trust.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.
- **@prerenderer/renderer-puppeteer**: Library for static pre-rendering.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Pre-render the project:
   ```
   pnpm run prerender
   ```
4. Build and pre-render together:
   ```
   pnpm run build:prerender
   ```
