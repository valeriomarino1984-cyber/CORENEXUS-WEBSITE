# Project Summary
CoreNexus Technology Solution is an IT support platform tailored for businesses in Rome. It provides a comprehensive suite of services including network management, cybersecurity, server support, VoIP solutions, project management, and IT process automation. The platform features an advanced ticket management system, integrated chat support, and a user-friendly admin dashboard, which streamline the handling of IT and cybersecurity needs. Recent updates have further enhanced the user experience with a multi-tenant architecture and improved SEO strategies.

# Project Module Description
The project consists of several key functional modules:
- **Homepage**: SEO-optimized introduction featuring a glitch effect on the main header.
- **Services Section**: Detailed overview of IT services including:
  - **IT Maintenance Contracts**: Information on maintenance packages, now enhanced with FAQ Schema JSON-LD markup.
  - **IT Consultancy Services**: Professional guidance for businesses, also updated with FAQ Schema.
  - **IT Process Automation**: Tailored automation solutions for operational efficiency.
  - **Project Management**: Tools for overseeing IT and cybersecurity projects.
  - **Networking and Infrastructure**: Setup and management services for networks.
  - **Cybersecurity Solutions**: Advanced measures utilizing Wazuh and Zabbix.
  - **Alarm and Surveillance Systems**: Collaborations with Hikvision, Ajax, and Risco for security solutions.
  - **Website and E-commerce Development**: Creation and management of online platforms.
  - **VoIP Solutions**: Support and configuration for VoIP systems.
- **Remote Support**: Page dedicated to remote assistance using UltraViewer.
- **Technologies**: Overview of utilized technologies.
- **Service Areas**: Details on coverage and response times.
- **Client Area**: Secure client login for ticket management.
- **Ticket Management**: Tools for processing support requests, now with multi-tenant support.
- **Admin Dashboard**: Management interface for tickets, users, and analytics.
- **User Management**: Admin features for managing users and companies with role-based access control.
- **Legal Documentation**: Compliance information.
- **Cookie Consent**: Management component for cookies.
- **Setup Page**: Feature for admin account creation.
- **SEO Enhancements**: Improved meta tags and content optimization.
- **Blog**: Articles on IT and cybersecurity, now including new AI-related content.
- **FAQ Sections**: Newly added FAQ sections for better user assistance, now integrated with JSON-LD Schema.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.
- **@prerenderer/renderer-puppeteer**: Library for static pre-rendering.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Pre-render the project:
   ```
   pnpm run prerender
   ```
4. Build and pre-render together:
   ```
   pnpm run build:prerender
   ```
