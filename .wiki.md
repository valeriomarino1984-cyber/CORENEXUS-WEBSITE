# Project Summary
CoreNexus is an advanced IT support platform tailored for businesses in Rome, offering comprehensive services including system administration, networking, cybersecurity, and 24/7 technical support. The platform enhances user experience through a sophisticated ticket management system, integrated WhatsApp chat support, and an intuitive admin dashboard. Recent updates have focused on improving user management, diagnostic logging, and resolving access issues related to ticket visibility through Row Level Security (RLS) policies.

# Project Module Description
The project includes several functional modules:
- **Homepage**: SEO-optimized introduction to IT services with interactive features.
- **Services Section**: Overview of IT services, including security systems and AI video verification.
- **Websites Section**: Information on responsive web design and e-commerce solutions.
- **Features Section**: Highlights of key service benefits.
- **Pricing Section**: Assistance packages available "on quote."
- **Brands Section**: Updated logos of technology partnerships.
- **Coverage Section**: Interactive map displaying service areas.
- **Contact Section**: Enhanced contact form with dynamic CAPTCHA and direct messaging to Telegram.
- **Client Area**: Secure login for clients to manage tickets.
- **Ticket Management**: Tools for ticket handling, real-time notifications, and support chat.
- **Admin Dashboard**: Module for managing tickets, viewing statistics, and responding to inquiries.
- **User Management**: System for creating, editing, and deleting users, assigning roles, and managing permissions.
- **Legal Pages**: Compliance documentation including Privacy Policy and Terms of Service.
- **Cookie Banner**: Component for cookie consent with enhanced visibility.
- **FreePBX Section**: Customizable FreePBX & Asterisk VoIP solutions priced "on quote."
- **UI Components**: Newly created components for improved user interaction.
- **WhatsApp Widget**: Integrated for chat support and ticket management.
- **Goals Section**: New section displaying company objectives.
- **SEO Enhancements**: Advanced meta tags, schema markup, and optimized content for better search engine visibility.
- **Header Component**: Updated with a modern design featuring smooth scrolling and enhanced navigation.
- **Setup Page**: Feature for creating admin accounts without email verification.
- **User Role Management**: SQL script for verifying and assigning user roles in the admin area.
- **Edge Function for User Creation**: Updated Edge Function to handle user creation with enhanced logging for diagnostics.
- **RLS Policy Management**: SQL script for managing Row Level Security policies for ticket visibility.
- **Debugging Tools**: New SQL script for diagnosing ticket access issues and detailed logging in the AdminTicketDetail component.
- **RLS Policy for Clients**: New SQL script to manage read permissions for the clients table, resolving access issues for admin users.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Lint the project:
   ```
   pnpm run lint
   ```
