# Project Summary
CoreNexus Technology Solution is an IT support platform designed for businesses in Rome, offering a comprehensive suite of services that includes network management, cybersecurity, server support, VoIP solutions, project management, and IT process automation. The platform features an advanced ticket management system, integrated chat support, and an intuitive admin dashboard, allowing businesses to effectively manage their IT and cybersecurity needs. Recent updates have improved user engagement with a reading progress indicator for blog articles and optimized the existing ticketing system.

# Project Module Description
The project consists of several functional modules:
- **Homepage**: SEO-optimized introduction to IT services with enhanced animations.
- **Services Section**: Overview of IT services, including:
  - **IT Maintenance Contracts**: Details on maintenance packages.
  - **IT Consultancy Services**: Expert advice for businesses.
  - **IT Process Automation**: Custom automation solutions for workflows.
  - **Project Management**: Tools for managing IT & Cyber Security projects.
  - **Networking and Infrastructure**: Services for network setup and management.
  - **Cybersecurity Solutions**: Advanced security measures using Wazuh and Zabbix.
  - **Alarm and Surveillance Systems**: Partnerships with Hikvision, Ajax, and Risco.
  - **Website and E-commerce Development**: Building and managing online platforms.
  - **VoIP Solutions**: Configuration and support for VoIP systems.
- **Remote Support**: Page for remote assistance using UltraViewer.
- **Technologies**: Overview of technologies utilized.
- **Service Areas**: Coverage and response times.
- **Client Area**: Secure login for clients to manage tickets.
- **Ticket Management**: Tools for handling support requests, with a fully implemented ticket system.
- **Admin Dashboard**: Management of tickets and analytics.
- **User Management**: Administering users and companies.
- **Legal Documentation**: Compliance information.
- **Cookie Consent**: Component for cookie management.
- **Setup Page**: Admin account creation feature.
- **SEO Enhancements**: Improved meta tags and content optimization.
- **Blog**: Section containing articles related to IT and cybersecurity, now featuring a reading progress indicator for enhanced user experience.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for development.
- **PostCSS**: Tool for transforming CSS.
- **ESLint**: Tool for identifying and fixing JavaScript code issues.
- **Supabase**: Backend as a service for authentication and database management.
- **Lenis.js**: Library for smooth scrolling.
- **Framer Motion**: Library for animations and gestures.
- **react-intersection-observer**: Library for scroll-triggered animations.
- **nodemailer**: Library for sending emails.
- **WhatsApp**: Integrated chat and ticketing solution.
- **react-helmet-async**: Library for managing document head and SEO.
- **@prerenderer/renderer-puppeteer**: Library for static pre-rendering.

# Usage
To get started with the project:
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Pre-render the project:
   ```
   pnpm run prerender
   ```
4. Build and pre-render together:
   ```
   pnpm run build:prerender
   ```
